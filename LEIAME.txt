                    UFPA Speech Recorder
                Versão 1.0, fevereiro de 2017

%%%%%%%%%%%%
% Windows
%%%%%%%%%%%%

# Dependências:
1) Python 2.7:
	https://www.python.org/downloads/release/python-2713/
	a) 32 bits --> Windows x86 MSI Installer
	b) 64 bits --> Windows x86-64 MSI Installer
2) PyQt4:
	https://sourceforge.net/projects/pyqt/files/PyQt4/PyQt-4.11.4/
	a) 32 bits --> PyQt4-4.11.4-gpl-Py2.7-Qt4.8.7-x32.exe
	b) 64 bits --> PyQt4-4.11.4-gpl-Py2.7-Qt4.8.7-x64.exe

# Execução:
Dois cliques sobre o arquivo "UFPA-Speech-Recorder" 

%%%%%%%%%%%%
% Linux
%%%%%%%%%%%%

# Dependências:
1) Python 2.7:
	sudo apt-get install python2.7
2) PyQt4:
	sudo apt-get install python-qt4 python-qt4-dev pyqt4-dev-tools
3) PyQt4:
	sudo apt-get install python-pyaudio (opcional, mas recomendado)

# Execução (dentro do diretório "src"):
1) make
ou
2) python -B UFPA-Speech-Recorder.py
